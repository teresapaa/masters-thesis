# masters_thesis 

Includes all code related to my master's thesis with the working title "GPU Accelerated Computing and Machine Learning for Economic Modeling"

Currently contains following directories:

neoclassical-model:
All the progress with the neoclassical model used as a practice model. The folder contains separate subfolders for execution time measurements and for plotting, to make sure plotting won't interefere with the execution time measurements. For neoclassical model, there are three solutions: valuefunction iteration, GPU accelerated valuefunction iteration and a reinforcement learning.

pension-model:
Next piece to work on, currently just contains a template to start building on the model.


Compiling and running:

Each script contains instructions on how to compile and run it.
